JoinAGame - Complete Dependencies List
=====================================

This document lists ALL dependencies required for the JoinAGame project to run properly on any local machine.

ROOT DIRECTORY DEPENDENCIES
===========================
Location: /package.json

Production Dependencies:
- eslint@^9.37.0          # Code linting and formatting
- sqlite3@^5.1.7         # SQLite database (legacy, not actively used)
- vite@^7.1.9            # Frontend build tool and dev server

Install Command: npm install

FRONTEND DEPENDENCIES (my-app/)
==============================
Location: /my-app/package.json

Production Dependencies:
- react@^19.2.0          # React framework
- react-dom@^19.2.0      # React DOM rendering
- react-router-dom@^6.30.1 # Client-side routing

Development Dependencies:
- @eslint/js@^9.36.0     # ESLint JavaScript configuration
- @types/react@^19.1.16  # TypeScript definitions for React
- @types/react-dom@^19.1.9 # TypeScript definitions for React DOM
- @vitejs/plugin-react@^5.0.4 # Vite plugin for React
- babel-plugin-react-compiler@^19.1.0-rc.3 # React compiler plugin
- eslint@^9.36.0         # Code linting
- eslint-plugin-react-hooks@^5.2.0 # React hooks linting
- eslint-plugin-react-refresh@^0.4.22 # React refresh linting

- vite@^7.1.9            # Frontend build tool

Install Command: cd my-app && npm install

BACKEND DEPENDENCIES (my-app/backend/)
=====================================
Location: /my-app/backend/package.json

Production Dependencies:
- express@^4.18.2        # Web framework for Node.js
- cors@^2.8.5            # Cross-Origin Resource Sharing middleware

Development Dependencies:
- nodemon@^3.0.1         # Auto-restart development server

Install Command: cd my-app/backend && npm install

COMPLETE INSTALLATION COMMANDS
==============================

For a fresh clone of the repository, run these commands in order:

1. Root Dependencies:
   cd mruhacks25-joinagame
   npm install

2. Frontend Dependencies:
   cd my-app
   npm install

3. Backend Dependencies:
   cd backend
   npm install

4. Start Backend:
   npm start

5. Start Frontend (in new terminal):
   cd my-app
   npm run dev

SYSTEM REQUIREMENTS
===================
- Node.js version 16.0.0 or higher
- npm version 8.0.0 or higher
- Git (for cloning)
- Modern web browser (Chrome, Firefox, Safari, Edge)

PORT REQUIREMENTS
=================
- Port 3001: Backend API server (must be available)
- Port 5173: Frontend development server (must be available)

NETWORK DEPENDENCIES
====================
- Internet connection for npm package downloads
- Local network access for frontend-backend communication
- No external API dependencies (fully self-contained)

FILE PERMISSIONS
================
- Read/write access to project directory
- Ability to create node_modules folders
- Ability to create/read/write data.json file

OPERATING SYSTEM COMPATIBILITY
==============================
- Windows 10/11
- macOS 10.15+
- Linux (Ubuntu 18.04+, CentOS 7+, etc.)

VERIFICATION COMMANDS
=====================
After installation, verify setup with:

Root:
npm list --depth=0

Frontend:
cd my-app && npm list --depth=0

Backend:
cd my-app/backend && npm list --depth=0

All commands should show the expected packages without errors.

NOTES
=====
- The project uses JSON file storage instead of a traditional database
- No database server installation required
- All dependencies are JavaScript/Node.js based
- The application is fully self-contained after installation
